# IBU cluster wiki

This wiki is generated with [MkDocs](https://www.mkdocs.org/), with the theme [Material](https://squidfunk.github.io/mkdocs-material/).

To host it locally, install MkDocs:
```bash
pip install mkdocs
```

and Material:
```
pip install mkdocs-material
```

Host it with:
```bash
mkdocs serve
```
